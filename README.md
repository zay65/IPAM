# IPAM
<h3>Repo for centrally managing IP address space</h3>

                                   




 


<div align="center">Launch "Server Manager"</div>
 

   <p align="center">
    <img src="https://github.com/zay65/IPAM/blob/78d7907fbc738a6e38c74fe0630e765e8cea8f9a/Server%20Manager.jpeg" alt="Sample Image"/>
  </p>


<div align="center">Select "Add Roles and Features"</div>
  <p align="center">
    <img src="https://github.com/zay65/IPAM/blob/78d7907fbc738a6e38c74fe0630e765e8cea8f9a/Add%20Roles%20and%20features%20.jpeg" alt="Sample Image"/>
  </p> 

<div align="center">Within the "Features" category, select and Install "IPAM"</div>
  <p align="center">
    <img src="https://github.com/zay65/IPAM/blob/78d7907fbc738a6e38c74fe0630e765e8cea8f9a/%20IP%20Address%20Management%20(PAM)%20Server.jpeg" alt="Sample Image"/>
  </p>

<div align="center">Close out Server Manager once IPAM Installs</div>
  <p align="center">
    <img src="https://github.com/zay65/IPAM/blob/78d7907fbc738a6e38c74fe0630e765e8cea8f9a/Install%20success%20.jpeg" alt="Sample Image"/>
  </p>

<div align="center">Open "PowerShell" and run "Get-WindowsFeature" command to see IPAM status</div>
